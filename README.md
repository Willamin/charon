# Charon

A command line application to manage project directories.

## Usage

- `$ charon version`
- `$ charon help`
- `$ charon list`
- `$ charon cherry`
- `$ charon new strawberry`
- `$ charon init orange`
- `$ charon clone pretzel git@github.com:pretzel/pretzel.git`
- `$ charon destroy apple`

## Development
Use `$ mix escript.build` to build the script.
